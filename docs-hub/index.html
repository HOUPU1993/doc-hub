Project Overview

This research project, led by Carole Turley Voulgaris with assistance from Houpu Li and Kamryn Mansfield, standardizes zoning and parcel data across the state of Texas and generates several prototypical building forms. It also develops a reusable R/Python toolkit that ingests these data feeds, assigns parcels to their zoning districts, produces compliant building footprints under zoning constraints, and estimates residential capacity.

Using that programming toolkit, we will model five prototypical housing forms on parcels deemed available under different availability definitions—ranging from all parcels to those sold within the past year—and calculate average accessibility to "destinations" by foot and by car for each of the twenty resulting scenarios.
